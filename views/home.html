Welcome to the Indextank demo

This is the same Poll application you built in the Django's tutorial, with searching features over Indextank engine.

To try it

  • Run python manage.py syncdb to create databases
  • Add some polls to your database. Go to Django's admin or hit this script
  • Go to the polls' home page and search you poll
  • Check polls/models.py and polls/views.py#search(request) to see how to integrate indextank with your application
