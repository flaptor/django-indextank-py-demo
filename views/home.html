Welcome to the Indextank demo

This is a simple django guestbook application with searching features using Indextank engine.

To try it

  • Run python manage.py syncdb to create database and tables
  • Add some entries to your guestbook. Go to Guestbook home anb fill the form or hit this script
  • Go to the Guestbook home and search your entry
  • Check guestbook/views.py to see how to integrate indextank with your application in seconds
